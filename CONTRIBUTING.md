# 贡献说明

欢迎提交 Issue、Discussion 或 Pull Request，帮助这套 Codex 13 插件项目流继续完善。

## 可以贡献什么

- 安装失败、插件不显示、运行报错等问题反馈
- 小白安装说明、使用说明、README 文案改进
- 新插件建议、现有插件流程优化建议
- 不同系统、不同 Codex 版本下的兼容性反馈
- 宣传图文、案例、教程、使用心得

## 提交 Issue 前

请尽量写清楚：

1. 你使用的是 Windows、macOS 还是 Linux。
2. 你下载的是哪个版本的安装包。
3. 你执行到哪一步失败了。
4. 失败时看到的完整提示。
5. 你希望最终达到什么效果。

## Pull Request 建议

- 改动尽量小而清楚。
- 不要提交真实密钥、账号、Cookie、token 或私有配置。
- 修改安装包相关内容后，请同步更新 `release/` 里的 zip。
- 修改插件说明后，请同步更新 README 或 Word 文档。

## 给技术小白的说明

如果你不会写代码，也可以直接开 Issue 或 Discussion，把你遇到的问题截图或描述出来。这个项目本身就是为了让技术小白更容易指挥 Codex 做完整项目。
